# Wyzer Logs Directory

This directory contains log files generated by Wyzer.

## Log Files

- `llamacpp_server.log` - Output from the embedded llama.cpp server (when using --llm llamacpp)

## Notes

- Log files are automatically created when needed
- The llama.cpp server log is appended to, not overwritten
- Check this log if the server fails to start or behaves unexpectedly
